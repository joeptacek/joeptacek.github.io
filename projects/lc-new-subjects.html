--- layout: project title: LC New Subjects ---

LC New Subjects is a Twitter bot for staying in touch with the latest subject heading updates from the Library of Congress (LC). LC releases these updates en masse, approximately once a month; LC New Subjects gathers these up and gradually Tweets its way through the heap.

My basic goal with this project is to help updates from LC reach a broader audience. LC publishes these updates online as HTML-formatted “approved lists.” The most obvious source for approved lists is a webpage linked from LC’s Cataloging and Acquisitions portal. What isn’t immediately obvious is that this webpage omits certain updates. Indeed, careful browsing uncovers a more complete index of approved lists on LC’s sister site, Classification Web. Searching for approved lists, one gets the sense these resources aren’t primarily intended for public consumption—they’re available, but not easily discoverable. This is consistent with the larger subject heading revision process, which technically allows for public input, but is far less welcoming to LC outsiders.

By using Twitter to rebroadcast subject heading updates, I hope to make it easier for librarians (and non-librarians) to monitor LC decision-making. I also hope to create space for ongoing reflection and conversation regarding LC subject heading terminology—since Twitter is a public forum, anyone can share or add comments on updates that resonate in one way or another, e.g., the recent decision to cancel the "Illegal aliens" subject heading.

Under the hood, LC New Subjects depends on two Python scripts—one for scraping the LC website, and another for programmatically sending out Tweets. I've documented and published all relevant code in two GitHub repositories, lc-scrape and lc-tweet. The lc-scrape repository also contains an archive of LC's approved lists, both in HTML and JSON format.
